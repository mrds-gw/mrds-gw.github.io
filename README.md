# [初见的永恒](https://mrds-gw.github.io/)

便利店的门叮咚一声滑开，凌晨三点的冷空气灌进来。值夜班的明浩抬起头，看到一个女孩站在门口，头发被雨淋湿，怀里抱着一只瑟瑟发抖的小狗。

“请问……有毛巾吗？还有牛奶，小狗能喝的。”女孩的声音很轻，带着犹豫。

那是他们的第一次对话。女孩叫素妍，小狗是她刚刚从路边纸箱里捡到的弃犬。明浩不仅给了她毛巾和牛奶，还分享了自己的夜宵——两盒微波炉加热的炒年糕。

“你总是在这里值夜班吗？”素妍问，小狗在她膝盖上舔着牛奶。

“一周三次。”明浩回答，“白天我在美术学院上课。”

“我也喜欢画画。”素妍的眼睛亮起来，“虽然我学的是植物学。”

那天晚上，素妍在便利店待了两个小时，直到雨停。他们聊了梵高笔下旋转的星空，聊了莫奈睡莲池中光的游戏，聊了植物叶脉中隐藏的数学之美。离开时，明浩在找零的纸条上画了一只微笑的小狗，下面写着：“它需要名字。”

一周后，素妍带着小狗再次出现，小狗脖子上挂着新项圈。“它叫‘午夜’，”她说，“因为是午夜遇见你的。”

这是爱情最简单的开始，像便利店暖光灯下的速食面，普通却温暖。他们开始相约在明浩不值夜的晚上，去24小时书店，去凌晨空荡的公园，去等待第一班地铁。

素妍教明浩认识城市中的野生植物——墙缝里的蕨类，行道树下的苔藓，废弃地块上倔强开花的野草。明浩则为素妍画下这些不起眼的生命，在他的素描本里，它们变得庄严而美丽。

“你知道吗？”一天夜里，素妍指着天空说，“有些花只在夜间开放，就像有些爱情只在特定时刻发生。”

明浩没有回答，只是牵起了她的手。那只叫“午夜”的小狗在他们脚边打转。

一年后，明浩的毕业作品在美术馆展出，是一系列名为《城市野性》的画作，主角是水泥缝隙中顽强的生命。素茹的植物图鉴也出版了，插图全部由明浩完成。开幕式上，他们并肩而立，手依然牵在一起。

如今，十五年过去了。明浩成了知名插画家，素茹在大学任教植物学。“午夜”已经去了彩虹桥，但他们又收养了两只猫，分别叫“晨曦”和“黄昏”。

有时，在忙碌的日常间隙，他们会不约而同地望向对方，同时说出：“去便利店吧。”

于是，在孩子们睡着的深夜，他们会悄悄出门，走到街角那家24小时便利店，买两盒微波炉炒年糕，坐在靠窗的位置。玻璃窗上反射出他们的面容——眼角有了细纹，头发添了银丝，但眼中的光芒依旧。

“还记得吗？”明浩总会问。

“永远记得。”素茹总会回答。

窗外，城市睡了又醒，季节轮转，星辰迁移。但在那方小小的、温暖的便利店里，时间仿佛静止在某个永恒的凌晨三点——当门叮咚一声滑开，一个抱着小狗的女孩走进来，而柜台后的男孩抬起头，看见了整个世界。

因为最美的爱情，有时就藏在最寻常的相遇里，像种子落在水泥地的缝隙中，看似不可能，却生根发芽，开出意想不到的绚烂花朵。
